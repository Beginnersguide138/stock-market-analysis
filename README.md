# Stock Market Analysis & Direct Multi-Step Forecasting Model

このプロジェクトは、指定した銘柄の株価（Open, High, Low, Close）の向こう5日間の推移を予測する、XGBoostベースのマルチステップ予測モデルです。

## モデルの概要

*   **アーキテクチャ**: Multi-step Direct Forecasting（マルチステップ直接予測）
    *   自己回帰モデル（AR）のように1日先の予測値を次の入力に使うのではなく、$t+1, t+2, \dots, t+5$ のそれぞれの日に対して個別のXGBoostモデルを学習させます。これにより、予測誤差が蓄積する（Snowball Effect）のを防ぎます。
*   **バックテスト手法**: ローリング・ウィンドウ方式
    *   特定の期間（例: 毎週末の金曜日）を基準日として、そこから未来5日間の予測を行います。これを1ヶ月分スライド（ローリング）させながら検証し、実際のチャートに重ね合わせて評価します。
*   **対象データ**: Yahoo Financeからの株価データ（過去最大10年分）、日経平均、ダウ平均、ローカルのUSD/JPY為替データ。

## 特徴量エンジニアリング (Feature Engineering)

ツリーベースのモデル（XGBoostなど）は、学習データで経験したことのない数値範囲（外挿領域）の予測が構造的に不可能であるという弱点を持っています。本モデルでは、以下の工夫によってこの問題を克服し、高い汎化性能を持たせています。

### 1. 非定常特徴量の相対化（Stationarity）
絶対価格（例: 株価が3000円、4000円といった数値）を直接入力するのではなく、すべてを**「当日の終値 (Close) を基準とした乖離率（比率）」**に変換しています。
*   **OHLC**: `(Open - Close) / Close` などに変換。
*   **テクニカル指標**: 一目均衡表（転換線、基準線、先行スパンA/B）、EMA（12, 26）、MACD、26日前ラグなどの絶対価格指標も、すべて当日のCloseで割って相対値化。
*   **出来高**: `Volume` の絶対値ではなく、前日比 `Vol_Change` を使用。
*   **ターゲット変数**: 予測対象（y）も未来の絶対価格ではなく、「現在の終値からのリターン（乖離率）」を予測させ、推論時にそれを絶対価格に再変換します。

### 2. 外部要因とモメンタム指標
対象銘柄単体のデータだけでなく、マクロ経済の動向を取り込むことで予測精度を向上させています。
*   **ダウ平均 (^DJI)**: 日本市場に先行して閉まるため、1日ずらした前日比リターンを使用。
*   **日経平均 (^N225)**: 日本市場全体の同日比リターンを使用。
*   **USD/JPY 為替**: 為替の当日の変動（リターン）を組み込み。

### 3. 【特殊仕様】1日先の情報カンニング (Cheat Features)
本モデルの最大の特徴として、**意図的なデータリーク（Leakage）**を活用した分析を行っています。
すべての特徴量（自銘柄のOHLC、テクニカル、為替、指数）について「1日先（翌日）のデータ」を `_cheat1d` 特徴量として作成し、モデルに入力しています。
*   **目的**: 「明日起こる最初のショック（初期変動）」をモデルに事前に教えることで、**そのショックがその後の2日目〜5日目の株価推移にどのような余波（アンカーリング効果）をもたらすか**を予測させるための特殊なシミュレーションモデルです。

## 学習プロセスの最適化

*   **欠損値 (NaN) の個別処理**: 5日先のターゲットが存在しない直近のデータにおいて、`dropna()` を全体に適用すると、学習に使えるはずの直近データが丸ごと欠落してしまいます。これを防ぐため、各ターゲット（1日先〜5日先）の学習ループ内で、必要な列のみに絞ってから `dropna()` を行う仕様に修正されています。
*   **高速化**: XGBoostの学習にヒストグラムベースのアルゴリズム (`tree_method='hist'`) を採用し、大量の特徴量とデータでも高速にモデル構築が可能になっています。
*   **非冪等な出力**: 予測結果を Plotly のインタラクティブな HTML チャートとして出力します。上書きを防ぐため、出力ファイル名には実行時のタイムスタンプが自動的に付与されます（例: `backtest_4443_YYYYMMDD_HHMMSS.html`）。

## 本モデルの実践的な強みと考察

複数銘柄（Sansan、ルネサス、ソフトバンクG、トヨタ自動車、レーザーテック等）でのバックテストを通じて、本モデル（1日カンニング＋マルチステップ予測）は、単なる「明日の株価当てゲーム」を超えた、以下の実践的な強みを持つことが確認されています。

### 1. 「ショックの余波」の正確なモデリング (Shock Propagation)
「明日の値動き（初期ショック）」をモデルに事前入力することで、その後の2日目〜5日目に発生する市場の力学を正確に描き出します。
例えば、1日目に大きな下落（あるいは為替の急変動）が起きた際、高ボラティリティ銘柄（レーザーテック等）において「そこから追証売りでさらに続落するのか」、あるいは「自律反発に向かうのか」といった、**初期ショックが引き起こす後続のモメンタムの変化**を高い精度でシミュレーションします。

### 2. トレンド転換点（底打ち・天井）の正確な価格帯の提示
本モデルの最大の価値は、「日々の細かな終値を当てること」よりも、**「株価の反転ポイント（Pivot）を正確な価格で予測できること」**にあります。
過去10年分の暴落や急騰のパターンを相対値（リターンベース）で学習しているため、「このレベルの初期下落が発生した場合、過去の統計的サポートライン（支持線）はどこになるか」を客観的に導き出します。これは実運用において、「どこに指値（押し目買い）や逆指値（損切り）を置くべきか」という具体的なトレード戦略に直結します。

### 3. 「慣性の法則」への非依存 (非線形パスの描画)
自己回帰モデル（前日の予測値を今日の入力に使う手法）の多くは、予測誤差が雪だるま式に拡大し、「昨日下がったから今日も下がる」という単調な線形トレンドを描きがちです。
一方、本モデルが採用する **Multi-step Direct Forecasting** は、$t+1$ から $t+5$ までを完全に独立したモデルで予測します。これにより、「2日目までは下落するが、3日目に底を打ち、4日目からV字反発する」といった、人間の心理（パニック売りからの買い戻し）を反映した**非線形な価格の軌跡（N字やV字の波形）**を描き出す能力に長けています。
