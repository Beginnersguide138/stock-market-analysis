# Micro Extremes Forecasting Model (ミクロ極値特化モデル)

このプロジェクトは、マクロ要因による終値の予測が困難であるという前提に立ち、**「その日のボラティリティ（1日の変動幅）と、どこまでヒゲを伸ばすか（極値）」** を精密に捉えることに特化したXGBoostベースの予測モデルです。

デイトレードや短期スイングトレードにおける**「最適な指値（押し目買いの下限）」**と**「最適な利確ライン（上値の限界）」**を客観的に提示することを目的としています。

## モデルのコア・コンセプト

### 1. 分位点回帰 (Quantile Regression) による限界値の予測
一般的な「平均的に最も誤差が少ない値（MSE/MAE最小化）」を予測するのではなく、**「分位点回帰」** を採用しています。
*   **高値 (High) の予測**: `quantile_alpha=0.90` (90パーセンタイル) に設定し、「上振れした際の上限（上ヒゲの先端付近）」を狙って予測します。
*   **安値 (Low) の予測**: `quantile_alpha=0.10` (10パーセンタイル) に設定し、「下振れした際の下限（下ヒゲの先端付近）」を狙って予測します。
これにより、モデルが保守的な「平均値」を出力するのを防ぎ、実際の荒い値動きのレンジをしっかりと包み込む予測帯を生成します。

### 2. 予測ターゲットの再定義
未来の「絶対価格」や「前日比リターン」ではなく、**「当日の始値(Open)を基準とした、高値・安値の乖離率」** をターゲット変数としています。
*   `Target_High = (High - Open) / Open`
*   `Target_Low = (Low - Open) / Open`
これにより、「終値は変わらなくても、日中どれくらい暴れるか」という純粋なボラティリティの質を直接学習させています。

### 3. ボラティリティと「ヒゲ」に特化した特徴量
株価の絶対値ではなく、銘柄固有の「荒っぽさ」を表現するための特徴量を重用しています。
*   **ATR (Average True Range)**: 過去の平均的な「1日の真の変動幅」。
*   **Bollinger Bands (ボリンジャーバンド)**: バンド幅（スクイーズとエクスパンション）および、バンド内の相対位置。
*   **過去のヒゲの傾向**: 過去5日間の「上ヒゲの比率」「下ヒゲの比率」の移動平均。その銘柄が現在、上ヒゲをつけやすい相場なのか、下ヒゲをつけやすい相場なのかを検知します。

### 4. 直近1年間のデータのみを学習（短期記憶への特化）
バックテストによる検証の結果、10年分や5年分の長期データを学習させるよりも、**「直近1年間のデータのみ」を学習させた方が、高値・安値の予測精度が圧倒的に高い（誤差が半減する）** ことが判明しました。
これは、市場のボラティリティの性質やアルゴリズムトレードの「クセ」の賞味期限が非常に短いためです。古い記憶を捨てることで、より「今の相場」に適合したシャープな予測が可能になっています。

## 含まれるスクリプト

*   **`predict_micro_extremes.py`**
    *   メインの予測スクリプトです。直近1年分のデータでモデルを学習し、起点日から向こう5日間の「予測始値」「予測安値(10%点)」「予測高値(90%点)」を出力します。また、予測帯を可視化したインタラクティブなHTMLチャートを生成します。
    *   実行例: `python predict_micro_extremes.py 4443.T`
*   **`evaluate_micro_extremes.py`**
    *   特定の基準日（例: 月末）までのデータでモデルを学習し、その翌週の「実際の高値・安値」と「予測値」を突き合わせて、誤差（MAE）を算出する評価用スクリプトです。
*   **`plot_historical_extremes.py`**
    *   指定期間（例: 過去1ヶ月間）の各営業日について、「前日までの1年間」で学習したモデルを使って当日の高値・安値を予測し続ける（Point-in-Time予測）シミュレーションを行い、実績のローソク足と予測帯を重ね合わせたチャートを生成します。モデルの「ヒゲ狩り」の精度を視覚的に確認できます。
*   **`evaluate_periods.py`**
    *   学習データの期間（10年、5年、3年、2年、1年）を変えた場合の予測精度を比較検証するためのバックテストスクリプトです。

## 実践での活用戦略

1.  **指値のセーフティーネット**: 予測された `Pred_Low` は、その日の強いサポートラインとして機能します。デイトレードの拾い場として、この付近に指値を置いておくことで、高値掴みのリスクを大幅に軽減できます。
2.  **欲張らない利確**: 予測された `Pred_High` は、その日の上値の限界点（上ヒゲの先端）になる可能性が高いです。ポジションを持っている場合は、このラインの少し手前で確実に利益確定を行うのが定石です。
3.  **マクロショックの警戒**: このモデルはあくまで「平常時のミクロな値動き」を前提としています。金利発表や暴落などの外部要因（ブラックスワン）が発生した場合は、予測下限をあっさりと下抜けるため、別軸でのリスク管理（逆指値など）が必須です。
